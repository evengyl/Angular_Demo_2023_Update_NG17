Demos 3 : Les chargement de vue différré

Supposons que vous ayez un blog et que vous souhaitiez charger paresseusement la liste des commentaires des utilisateurs. Actuellement, vous devez utiliser tout en gérant toute la complexité des nettoyages, en gérant les erreurs de chargement, en affichant un espace réservé, etc. La prise en charge de divers cas particuliers peut entraîner un code non trivial, qui sera difficile à tester et à déboguer.ViewContainerRef Les nouvelles vues différables vous permettent de charger paresseusement la liste des commentaires et toutes leurs dépendances transitives avec une seule ligne de code déclaratif :

@defer (on immediate; prefetch on idle) { @for (item of listItem; track item.id) { } } @loading { Loading… } @error { Loading failed :( } @placeholder { }
